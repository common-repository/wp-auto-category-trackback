== Description ==

This plugin provides a new function that send trackback automatically when a post published in specific category like MT.

You can set trackback url proper to Category.

When a post published in the specific category, this plugin will send trackback automatically.

Setting multiple urls and publishing multi-category post are also available.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

== Uninstallation ==

Open any specific category editing form, then put "**uninstall**"(without quotation) into trackback url box and update category. Setting of this plugin will be deleted.
